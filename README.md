# Tip Calculator

## Project Overview
Developed a Python-based Tip Calculator that calculates the total bill amount, tip percentage, and individual share for each person.

## Features
- Calculates tip amount automatically
- Splits the bill among multiple people
- User-friendly command-line interface
- Accurate bill distribution

## Technologies Used
- Python

## Learning Outcomes
- User Input Handling
- Arithmetic Operations
- Variables and Data Types
- Basic Problem Solving in Python
